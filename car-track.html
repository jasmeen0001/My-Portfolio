CAR-TRACK

The Car Sales Management System is a Python-based application designed to simplify vehicle sales transactions by automating cost calculations and receipt generation. Built using Python and Tkinter for the graphical user interface, the system allows users to input customer details such as name, address, postcode, and telephone number. It includes a dropdown menu for selecting a car, entering its cost, and specifying mileage, along with an option to trade in a vehicle. Customers can also choose additional features like modifications, a stereo system, leather interior, custom details, and a GPS system. The system automatically calculates VAT, discounts, and taxes, providing an accurate total cost. With integrated buttons for calculating totals, generating receipts, resetting inputs, and exiting the system, the interface is user-friendly and efficient. This project enhances accuracy in car sales transactions while improving customer service and operational efficiency.
